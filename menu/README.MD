🍹 Kafenin Ətraflı Menyusu

Bu bölmədə "Türkanın Ləziz Dünyası" kafesində təqdim olunan xüsusi təamlar və imzalı içkilərin siyahısı yer alır.

🪄 Magic Drinks (Sehirli İçkilər)
Kafemizin vizual olaraq göz oxşayan, rəngarəng və xüsusi qarışıqlardan ibarət imzalı içkiləri:
1. Göy Qurşağı Kokteyli — Təbii meyvə siropları və xüsusi bitki çayları ilə rəngini dəyişən sehrli soyuq içki.
2. Lavanda Möcüzəsi — Təzə lavanda ekstraktı və südün mükəmməl harmoniyası.
3. Qızılı Espresso — Üzərində yeməli qızıl hissəcikləri olan estetik və sərt qəhvə.

🥐 Tasty Foods (Ləziz Təamlar)
Hər gün təzə bişən, fransız mətbəxinin zərifliyini daşıyan şirniyyat və pastry məhsullarımız:
 Klassik Fransız Kruassanı — Kərə yağı ilə hazırlanan, qat-qat və xırtıldayan daddadır.
 Şokoladlı Ekler şousu — İçi xüsusi krem, üstü isə Belçika şokoladı ilə örtülmüş unudulmaz desert.
 Moruqlu Təbəssüm — Təzə moruqlar və yüngül kremlə hazırlanan estetik mini tort.
