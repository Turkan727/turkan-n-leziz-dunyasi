# turkan-n-leziz-dunyasi
Zərif şirniyyatların, xüsusi bişmələrin və sehirli içkilərin unikal məkanı. Bu layihə "Petit Türkan" kafesinin brendinq, menyu və konseptual planlaşdırılmasını əhatə edir.

## 📌 Konsept Haqqında
Kafemiz müştərilərə vizual olaraq estetik, dad olaraq isə unudulmaz təcrübə bəxş etmək üçün yaradılmışdır. Hər bir detal xüsusi qayğı ilə düşünülmüşdür.

## 🍹 Menyu Kateqoriyaları
* Magic Drinks: Xüsusi reseptlərlə hazırlanan rəngarəng və sehirli imzalı içkilər.
* Tasty Foods: Hər səhəri gözəlləşdirən təzə kruassanlar, pastry məhsulları və unikal desertlər.

## 🚀 Marketinq və Brend Strategiyası
* Estetik vizual kontent planı (Sosial media yönümlü).
* Müştəri loyallığı proqramları və xüsusi hədiyyə kartları.
